# for-site image set

This folder is a curated copy set for quick site selection.
Original files remain in `images/`.

## Folders
- `logos/`: brand and product logos
- `team/`: member photos
- `architecture/`: hero/visual candidates
- `business/`: business competition slide assets
- `global-network/`: partner/global network assets
- `core-services/`: service domain assets
- `company-profile/`: company profile slide assets
- `timeline/1985`, `timeline/1999`, `timeline/2005`, `timeline/2024`: year-grouped timeline assets

## Notes
- File names in this folder are partly normalized for readability.
- Safe to review and pick from here without touching originals.
