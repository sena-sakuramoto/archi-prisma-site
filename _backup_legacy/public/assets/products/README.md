## 楽々省エネ計算ロゴ

- 追加先ファイル名: `rakuraku-logo.png`
- 参照先: `/assets/products/rakuraku-logo.png`

`src/components/Products.astro` でファイル存在チェックをしているため、
`rakuraku-logo.png` を配置すると自動で表示されます。
